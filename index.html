Projects

SquadUp

A Progressive Web App Startup

Created a progressive web app (PWA) that facilitates large group decision making. It allows groups to enter preferences and locations, curates and displays results for resturant recommendations based on Yelp's API, and allows for group voting in multiple modes. The application is available at: squadup.app. Since it is a PWA it can be used on a computer browser and saved to iOS and Android home screens on phones, where it functions like a Native app and does not take up storage space.

Program Language Translation Research

Paper Accepted to ICML's 2018 Workshop NAMPI

Researched on program language translation using a grammar driven tree-to-tree encoder/decoder model that uses tree-based neural networks to improve the accuracy of translation. Paper available at: https://arxiv.org/abs/1807.01784

Autonomous Vehicle

A gold code emitting and detecting robot

Constructed an autonomous vehicle that detects and emits gold codes, navigates paths based on the color of the floor, and can gain points for the team on a competition field

BeautyStroke App
